## 🏗 System Architecture

The application is built using:
- JavaFX for the user interface
- Java for backend logic
- File-based storage system
- Email integration for notifications

Key Components:
- `HelloApplication.java`: Main application entry point
- `Home.java`: Main dashboard implementation
- `Admin.java`: Administrative interface
- `TaskClient.java`: Task management logic
- `FriendClient.java`: Friend system implementation
- `EmailSender.java`: Email notification system

## 🚀 Installation

### Prerequisites
- Java JDK 11 or higher
- JavaFX SDK
- IDE (preferably IntelliJ IDEA)

### Setup Steps
1. Clone the repository
2. Open the project in your IDE
3. Configure JavaFX SDK
4. Build and run the application

## 📖 Usage Guide

### For Regular Users
1. **Login/Registration**
   - Create a new account or login with existing credentials
   
2. **Task Management**
   - Create new tasks
   - Set priorities and due dates
   - Track task progress
   
3. **Friend System**
   - Send friend requests
   - Accept/reject incoming requests
   - Share tasks with friends

### For Administrators
1. **User Management**
   - View all users
   - Manage user permissions
   - Monitor user activity

2. **System Management**
   - View system analytics
   - Manage application settings
   - Generate reports

## 🔧 Technical Details

### Key Classes
- `Task.java`: Task data model
- `Friend.java`: Friend system model
- `TaskClient.java`: Task operations handler
- `OverdueTaskNotifier.java`: Notification system
- `CelebrationAnimation.java`: UI animations

### Features Implementation
- Task sharing using `SharedTasksPage.java`
- Friend management through `FriendsPage.java`
- Request handling in `RequestsPage.java`
- Analytics with `TaskAnalytics.java`
